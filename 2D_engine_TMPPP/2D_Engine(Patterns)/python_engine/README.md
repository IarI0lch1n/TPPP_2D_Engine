## Run UI demo (pygame-ce)

From repo root (folder that contains `python_engine/`):

- Install:
  - `pip install -r python_engine/requirements.txt`

- Run:
  - `python -m python_engine.apps.demo_pygame`
  - or run `python_engine/apps/demo_pygame.py` from IDE

TAB switches UI theme (Abstract Factory in action).
Buttons spawn entities via Builder into the World.
